# HOW TO START
To execute the project:
1) Install anaconda
2) bash ./conda_start.sh
3) pip install -r requirements.txt

# Project Components:

|            file|                                 function|

|test_Handlers.py|                      Testing of Handlers|
|      Storage.py| Interfaces for saving objects to storage|
